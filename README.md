## How to use

You can find dist in [docs/.vuepress/public/css](docs/.vuepress/public/css).

### CDN

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@yunyoujun/ak-ui">
```

### NPM or YARN

```sh
npm install @yunyoujun/ak-ui
# or
yarn add @yunyoujun/ak-ui
```
